ОПИСАНИЕ ФАЙЛОВ:

Board.py - подготавливает файл с головоломкой к работе в виде "доски" и предоставляет ее для Solverer
Combination.py - содержит в себе все возможные значения для определенного количества блоков и определенной суммы
NoneBlock.py - класс, описывающий блоки, которые не имеют значений
SumBlock.py - класс, описывающий блоки, которые содержат значения сумм
ValueBlock.py - класс, описывающий блоки, в которые записываются значения
Solverer.py - класс, который решает головоломку


ПРОЦЕСС РЕШЕНИЯ:

В классе Board происходит подготовка исходного файла для работы.
В классе Solverer происходит решение головоломки.
Решение реализуется тремя методами:
1) Поиск простых комбинаций
	Поиск блоков, с единственным возможным решением.
2) Поиск дополнений до суммы
	Поиск блоков, которые принадлежат ряду со свойством: известны все значения кроме одного.
	В пустое значение ставиться разность между искомой суммой и суммой известных значений.
3) Перебор
	В список записываются все возможные варианты решений.
Метод Checker выполняет проверку решения методом перебора сумм. Если Checker вернул True,
то проверяемое решение является верным.
